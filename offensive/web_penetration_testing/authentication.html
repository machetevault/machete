::::: [IP-BASED PROTECTION]

Some applications have some restrictions about IP-based / Geolocation and more, one way to by-pass or attempt to 
by-pass the IP-based restriction is to use the header, "X-Forwarded-For".





TIME DELAY

For this testing it is important to use different users, if possible authenticated/valid session and non valid 
session. Comparing responses will be needed. 

Check if response of invalid user takes more time than a valid user and viceversa, that can help to discover or
associate valid authentication credentials.
