CURL
DESCRIPTION: Tool used to make HTTP requests, commonly used to fingerprint a website. 

C1: curl -I TYPE_URL
SAMPLE: curl -I https://machetevault.github.io

-I = headers
-L = follow redirect

NMAP
DESCRIPTION: Tool used to make scan or fingerprint devices.  

C1: nmap -p[port] -A URL
SAMPLE: nmap -p 443,80 -A machetevault.github.io

-A = print all
-p = port
