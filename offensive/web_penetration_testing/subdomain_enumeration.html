[SUBDOMAIN ENUMERATION]







AMASS
DESCRIPTION:

C1: amass enum -d URL
SAMPLE: amass enum -d machetevault.com
SAMPLE: amass enum -d machetevault.com > /path/to/file.txt
	
-d = domain
> = output content

 





WEB TOOLS
DESCRIPTION:

crt.sh 

NOTE:  Wildcard search on crt.sh, %.URL
SAMPLE: %.machetevault.com
	




SUBFINDER
DESCRIPTION:

C1: subfinder -d URL
SAMPLE: subfinder -d machetevault.com
SAMPLE: subfinder -d machetevault.com -o /path/to/file.txt
	
-d = domain
-o = output content

 





GOWITNESS
DESCRIPTION:

C1: gowitness file -f demo.txt -P folder/to/store --no-http
SAMPLE: gowitness file -f demo.txt -P /usr/store/ --no-http
	
--no-http = ignore HTTP requests
-P = folder to store images

 





SUBLIST3R
DESCRIPTION:

$ sublist3r -d URL
$ sublist3r -d machetevault.com

flags:

-d: domain
