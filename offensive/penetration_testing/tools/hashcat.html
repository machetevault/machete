[HASHCAT]






NOTE: This tool can be used to crack different hash types, also, this tool has better performance using GPU instead
of CPU. 

COMMON SYNTAX: hashcat -m value -a value hashfile wordlist

-m: hash type
-a: attack mode
-h: help

SAMPLE: hashcat -m 0 -a 0 md5.txt rockyou.txt
  
NOTE: Help can be used to check the different values for the "HASH MODES". 

Section, - [ Hash modes ] -

MD5: 0
SHA1: 100
SHA256: 1400
SHA512: 1700
NTLM: 1000
