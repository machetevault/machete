[NMAP]






NOTE: 

normal command:

$ nmap -sC -sV -p- -T4 -oN filename.txt IP_ADDR

flags:

-sC: run with scripts NSE
-sV: 
-p-: run against all ports / 65535
-T4: make it faster
-oN: write output to file

network discover command:

$ nmap -sn NETWORK_IP/MASK
$ nmap -sn 172.17.9.0/24
